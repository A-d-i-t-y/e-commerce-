# E-Commerce Project

A full-featured e-commerce application built with EverShop.

## Features
- Product management
- Shopping cart
- Order management
- Customer accounts
- Payment integration

## Tech Stack
- Node.js
- React
- PostgreSQL

## Getting Started

### Installation
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```
